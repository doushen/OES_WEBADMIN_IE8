iframeWin
=========

iframeWin For Easy UI. 为 Easy UI 扩展的支持IFrame插件


在一个项目中用了Easy UI，但是发现里面的 Dialog 、Window、Messager 弹窗都不支持IFrame的方式，
于是自己动手基于Easy UI实现了一个插件。

<br/>
目前Easy UI仍然使用GPL开源协议。
<br/>

插件：jquery.iframeWin.js

开发者：李锡远
